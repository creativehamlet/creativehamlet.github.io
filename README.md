## Installation



### Install with Git

Clone the repo.
```
git clone https://gitlab.com/amoeb/amoeb.io.git
```

Go into the repo.
```
cd amoeb.io
```

Install the dependencies.
```
npm i
```

Build for dev and run on local server.
```
npm run serve
```
